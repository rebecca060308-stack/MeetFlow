TeamSync

대학생 팀 프로젝트 일정 조율 및 업무 관리 통합 플랫폼

프로젝트 개요

회의 시간 추천과 칸반보드 기반 업무 관리를 통합 제공하는 협업 플랫폼

주요 기능

  • 회원가입 / 로그인 (JWT 인증)
  • 팀 생성 및 초대
  • 공통 가능 시간 자동 분석
  • 회의 시간 Top 3 추천
  • 칸반보드 업무 관리
  • 진척도 자동 계산

사용자 스토리

팀장은 팀을 관리하고 회의 시간을 빠르게 결정할 수 있다.

팀원은 자신의 일정과 업무를 효율적으로 관리할 수 있다.

ERD

User - Team - TeamMember - Schedule - Task

기술 스택

  • Frontend : React
  • Backend : Spring Boot
  • Database : MySQL
  • Authentication : JWT

기대 효과

  • 회의 일정 조율 효율 향상
  • 협업 생산성 증가
  • 진행 상황 가시성 확보
