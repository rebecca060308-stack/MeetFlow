# TeamSync
## 대학생 팀 프로젝트 일정 조율 및 업무 관리 통합 플랫폼

---

# 1. 프로젝트 개요

## 1.1 프로젝트명
**TeamSync**

## 1.2 프로젝트 소개
대학생들은 팀 프로젝트를 진행할 때 회의 시간을 조율하고 역할을 분배하며 진행 상황을 공유하는 과정에서 많은 어려움을 겪는다. 현재는 카카오톡, 구글 캘린더, 노션, 트렐로 등 여러 서비스를 함께 사용해야 하는 불편함이 존재한다.

본 프로젝트는 일정 조율과 업무 관리를 하나의 플랫폼에서 해결할 수 있는 통합 협업 서비스를 제공하는 것을 목표로 한다.

---

# 2. 배경 및 필요성

## 문제점

1. 회의 시간 조율에 많은 시간 소모
2. 역할 분담이 명확하지 않음
3. 업무 진행 상황 파악 어려움
4. 여러 플랫폼 사용으로 인한 비효율

## 해결 방안

- 공통 가능 시간 자동 계산
- 회의 시간 추천 기능
- 칸반보드 기반 업무 관리
- 진척도 자동 계산

---

# 3. 사용자 스토리

### 팀장

- 팀을 생성하고 팀원을 초대하고 싶다.
- 팀원들의 가능한 시간을 한 번에 확인하고 싶다.
- 프로젝트 진행률을 확인하고 싶다.

### 팀원

- 가능한 시간을 쉽게 등록하고 싶다.
- 자신에게 할당된 업무를 확인하고 싶다.
- 업무 진행 상황을 업데이트하고 싶다.

---

# 4. 기능 요구사항

## 회원가입 및 로그인

### 회원가입
- 이메일 입력
- 비밀번호 입력
- 닉네임 입력
- 이메일 중복 확인
- 비밀번호 유효성 검사

### 로그인
- 이메일 인증
- 비밀번호 검증
- JWT 발급

### 팀 생성 및 초대
- 팀 생성
- 초대 코드 생성
- 팀 참가

---

## 일정 조율 시스템

### 개인 일정 등록
- 캘린더 UI
- Drag & Drop 입력
- 수정 및 삭제

### 공통 시간 분석
- 팀원 일정 통합
- 공통 빈 시간 계산
- 색상 시각화

### 회의 시간 추천
- 공통 시간 추출
- 우선순위 계산
- Top 3 추천

---

## 업무 관리 시스템

### 업무 생성
- 업무명
- 담당자
- 마감일
- 설명

### 업무 상태 변경
- To Do
- In Progress
- Done

### 진척도 계산
진척도 = (완료 업무 수 / 전체 업무 수) × 100

# 10. 결론

TeamSync는 대학생 팀 프로젝트에서 가장 큰 문제인 일정 조율과 업무 관리를 통합적으로 해결하는 웹 서비스이다. 이를 통해 협업 효율성을 높이고 프로젝트 진행 과정을 체계적으로 관리할 수 있다.
